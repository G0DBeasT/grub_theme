# Contributing to This Theme
Welcome and thanks for your interest in contributing to this GRUB theme!

## Support for Other Resolutions
Despite GRUB's limited font rendering, I've tried a different font for minimalistic view. I did this by overlaying a screenshot of the main menu over a screenshot from [grub2-theme-preview]. However, I do not have access to higher resolutions than 1920x1080. If you decide to add support for another resolution, please make sure to check whether the text aligns with the original. Please open an issue before doing this, so two people won't be working on the same solution.

## Optimizations and Issues
Feel free to pull request or open an issue to add improvements to this repository like install scripts or general improvements to the organization of this repository.

## Documentation
Add installation guides for other distros.
